# Astrophysicist Portfolio Website

A stunning, interactive portfolio website for astrophysicists featuring a constellation-based navigation system. Built with React, Framer Motion, and modern web technologies.

## Features

- **Constellation Navigation**: Interactive star map for navigating between sections
- **Day/Night Theme Toggle**: Switch between dark space theme and light theme
- **Animated Star Field**: Dynamic background with twinkling stars
- **Responsive Design**: Works perfectly on all devices
- **Smooth Animations**: Powered by Framer Motion
- **Modern UI**: Glass-morphism effects and gradient designs

## Sections

- **Home**: Welcome page with hero section and statistics
- **About Me**: Personal information, contact details, and message form
- **Publications**: Research papers with tabs for recent and selected works
- **Gallery**: Astrophotography gallery with category filters and modal view
- **Conferences**: Speaking engagements and presentations
- **Events**: Public lectures, workshops, and outreach activities
- **CV Download**: Professional experience, education, and achievements

## Technologies Used

- React 18
- Framer Motion (animations)
- React Router (navigation)
- Lucide React (icons)
- CSS3 with custom properties
- Modern JavaScript (ES6+)

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd astrophysicist-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Customization

### Personal Information

Replace placeholder content in the following files:
- Update `[Your Name]` throughout the codebase
- Replace `[University Name]` with your institution
- Update contact information in `src/pages/About.js`
- Modify research interests and publications

### Images

Replace the placeholder images in the Gallery section:
- Update image URLs in `src/pages/Gallery.js`
- Add your own astrophotography or research images

### Content

Customize the content in each page:
- **Home**: Update statistics and description
- **About**: Personal journey and research interests
- **Publications**: Add your actual publications with DOIs
- **Conferences**: Update with your speaking engagements
- **Events**: Add your outreach activities
- **CV**: Update with your professional experience

### Styling

The website uses CSS custom properties for easy theming:
- Colors are defined in `src/index.css`
- Component-specific styles in respective CSS files
- Responsive breakpoints for mobile optimization

## Building for Production

This builds the app for production to the `build` folder.
